Plain Summary

The project studies the same live inbox emails in more than one form. First, it checks the plain message text. Then it converts the same emails into metadata-aware form — where links, action words, financial cues, and message style are measured. A final combined learner then compares both branches together. This gives stronger live performance than relying on a single raw-text path.
